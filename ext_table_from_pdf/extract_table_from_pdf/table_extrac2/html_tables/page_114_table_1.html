Value Description
<hue> specified in degrees around the color wheel (without units), where 0° is red,\r60° is yellow, 120° is green, 180° is cyan, 240° is blue, 300° is magenta, and\r360° is red
<saturation> percentage where 0% is fully desaturated (grayscale) and 100% is fully\rsaturated (vividly colored)
<lightness> percentage where 0% is fully black and 100% is fully white
<alpha> a number from 0 - 1 where 0 is fully transparent and 1 is fully opaque
