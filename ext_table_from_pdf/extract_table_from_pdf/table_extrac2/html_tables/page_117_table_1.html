Parameter Details
counter-\rname This is the name of the counter that needs to be created or incremented or\rprinted. It can be any custom name as the developer wishes.
integer This integer is an optional value that when provided next to the counter name\rwill represent the initial value of the counter (in counter-set, counter-reset\rproperties) or the value by which the counter should be incremented (in\rcounter-increment).
none This is the initial value for all 3 counter-* properties. When this value is used\rfor counter-increment, the value of none of the counters are affected. When this\ris used for the other two, no counter is created.
counter-\rstyle This specifies the style in which the counter value needs to be displayed. It\rsupports all values supported by the list-style-type property. If none is used\rthen the counter value is not printed at all.
connector-\rstring This represents the string that must be placed between the values of two\rdifferent counter levels (like the "." in "2.1.1").
