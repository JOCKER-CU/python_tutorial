Value Effect
none Hide the element and prevent it from occupying space.
block Block element, occupy 100% of the available width, break after element.
inline Inline element, occupy no width, no break after element.
inline-\rblock Taking special properties from both inline and block elements, no break, but can\rhave width.
inline-\rflex Displays an element as an inline-level flex container.
inline-\rtable The element is displayed as an inline-level table.
grid Behaves like a block element and lays out its content according to the grid\rmodel.
flex Behaves like a block element and lays out its content according to the flexbox\rmodel.
inherit Inherit the value from the parent element.
initial Reset the value to the default value taken from behaviors described in the HTML\rspecifications or from the browser/user default stylesheet.
table Behaves like the HTML table element.
table-\rcell Let the element behave like a <td> element
table-\rcolumn Let the element behave like a <col> element
table-row Let the element behave like a <tr> element
list-item Let the element behave like a <li> element.
