Parameter Details
transition-\rproperty The specific CSS property whose value change needs to be transitioned (or)\rall, if all the transitionable properties need to be transitioned.
transition-\rduration The duration (or period) in seconds (s) or milliseconds (ms) over which the\rtransition must take place.
transition-\rtiming-\rfunction A function that describes how the intermediate values during the transition\rare calculated. Commonly used values are ease, ease-in, ease-out, ease-in-\rout, linear, cubic-bezier(), steps(). More information about the various\rtiming functions can be found in the W3C specs.
transition-\rdelay The amount of time that must have elapsed before the transition can start.\rCan be specified in seconds (s) or milliseconds (ms)
