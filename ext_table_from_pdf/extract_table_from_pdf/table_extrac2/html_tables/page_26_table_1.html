Function/Parameter Details
skewY(y) Vertical shear mapping distorting each point of an element by a\rcertain angle in the vertical direction
matrix() Defines a 2D transformation in the form of a transformation matrix.
angle The angle by which the element should be rotated or skewed\r(depending on the function with which it is used). Angle can be\rprovided in degrees (deg), gradians (grad), radians (rad) or turns (turn\r). In skew() function, the second angle is optional. If not provided,\rthere will be no (0) skew in Y-axis.
length-or-percentage The distance expressed as a length or a percentage by which the\relement should be translated. In translate() function, the second\rlength-or-percentage is optional. If not provided, then there would be\rno (0) translation in Y-axis.
scale-factor A number which defines how many times the element should be\rscaled in the specified axis. In scale() function, the second scale-\rfactor is optional. If not provided, the first scale-factor will be applied\rfor Y-axis also.
