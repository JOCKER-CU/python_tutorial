Parameters Details
inset by default, the shadow is treated as a drop shadow. the inset keyword draws\rthe shadow inside the frame/border.
offset-x the horizontal distance
offset-y the vertical distance
blur-radius 0 by default. value cannot be negative. the bigger the value, the bigger and\rlighter the shadow becomes.
spread-\rradius 0 by default. positive values will cause the shadow to expand. negative values\rwill cause the shadow to shrink.
color can be of various notations: a color keyword, hexadecimal, rgb(), rgba(),\rhsl(), hsla()
