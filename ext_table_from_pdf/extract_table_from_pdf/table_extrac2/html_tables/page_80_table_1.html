Parameter Details
clip-source A URL which can point to an inline SVG element (or) an SVG element in an\rexternal file that contains the clip path's definition.
basic-shape Refers to one among inset(), circle(), ellipse() or polygon(). Using one of\rthese functions the clipping path is defined. These shape functions work\rexactly the same way as they do in Shapes for Floats
clip-\rgeometry-box This can have one among content-box, padding-box, border-box, margin-box,\rfill-box, stroke-box, view-box as values. When this is provided without any\rvalue for <basic-shape>, the edges of the corresponding box is used as the\rpath for clipping. When used with a <basic-shape>, this acts as the\rreference box for the shape.
mask-\rreference This can be none or an image or a reference URL to a mask image source.
repeat-style This specifies how the mask should be repeated or tiled in the X and Y axes.\rThe supported values are repeat-x, repeat-y, repeat, space, round, no-repeat.
mask-mode Can be alpha or luminance or auto and indicates whether the mask should be\rtreated as a alpha mask or a luminance mask. If no value is provided and the\rmask-reference is a direct image then it would be considered as alpha mask\r(or) if the mask-reference is a URL then it would be considered as luminance
