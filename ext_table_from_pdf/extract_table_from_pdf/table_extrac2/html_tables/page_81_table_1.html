bg-size This represents the size of each mask-image layer and has the same syntax\ras background-size. The value can be length or percentage or auto or cover\ror contain. Length, percentage and auto can either be provided as a single\rvalue or as one for each axis.
compositing-\roperator This can be any one among add, subtract, exclude, multiply per layer and\rdefines the type of compositing operation that should be used for this layer\rwith those below it. Detailed explanation about each value is available in the\rW3C Specs.
