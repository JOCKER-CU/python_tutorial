Modifier Description Example
FM prefix fill mode (suppress leading ze-\rroes and padding blanks) FMMonth
TH suffix upper case ordinal number suffix DDTH, e.g., 12TH
th suffix lower case ordinal number suffix DDth, e.g., 12th
FX prefix fixed format global option (see\rusage notes) FX Month DD Day
TM prefix translation mode (use localized\rday and month names based on\rlc_time) TMMonth
SP suffix spell mode (not implemented) DDSP
