IO Wait Event Description
SnapbuildSync Waiting for a serialized historical catalog snapshot\rto reach durable storage.
SnapbuildWrite Waiting for a write of a serialized historical cata-\rlog snapshot.
TimelineHistoryFileSync Waiting for a timeline history file received via\rstreaming replication to reach durable storage.
TimelineHistoryFileWrite Waiting for a write of a timeline history file re-\rceived via streaming replication.
TimelineHistoryRead Waiting for a read of a timeline history file.
TimelineHistorySync Waiting for a newly created timeline history file to\rreach durable storage.
TimelineHistoryWrite Waiting for a write of a newly created timeline his-\rtory file.
TwophaseFileRead Waiting for a read of a two phase state file.
TwophaseFileSync Waiting for a two phase state file to reach durable\rstorage.
TwophaseFileWrite Waiting for a write of a two phase state file.
VersionFileWrite Waiting for the version file to be written while cre-\rating a database.
WALBootstrapSync Waiting for WAL to reach durable storage during\rbootstrapping.
WALBootstrapWrite Waiting for a write of a WAL page during boot-\rstrapping.
WALCopyRead Waiting for a read when creating a new WAL seg-\rment by copying an existing one.
WALCopySync Waiting for a new WAL segment created by copy-\ring an existing one to reach durable storage.
WALCopyWrite Waiting for a write when creating a new WAL\rsegment by copying an existing one.
WALInitSync Waiting for a newly initialized WAL file to reach\rdurable storage.
WALInitWrite Waiting for a write while initializing a new WAL\rfile.
WALRead Waiting for a read from a WAL file.
WALSenderTimelineHistoryRead Waiting for a read from a timeline history file dur-\ring a walsender timeline command.
WALSync Waiting for a WAL file to reach durable storage.
WALSyncMethodAssign Waiting for data to reach durable storage while as-\rsigning a new WAL sync method.
WALWrite Waiting for a write to a WAL file.
