IPC Wait Event Description
AppendReady Waiting for subplan nodes of an Append plan\rnode to be ready.
ArchiveCleanupCommand Waiting for archive_cleanup_command to com-\rplete.
ArchiveCommand Waiting for archive_command to complete.
BackendTermination Waiting for the termination of another backend.
BackupWaitWalArchive Waiting for WAL files required for a backup to be\rsuccessfully archived.
BgWorkerShutdown Waiting for background worker to shut down.
BgWorkerStartup Waiting for background worker to start up.
BtreePage Waiting for the page number needed to continue a\rparallel B-tree scan to become available.
BufferIO Waiting for buffer I/O to complete.
CheckpointDone Waiting for a checkpoint to complete.
CheckpointStart Waiting for a checkpoint to start.
ExecuteGather Waiting for activity from a child process while ex-\recuting a Gather plan node.
HashBatchAllocate Waiting for an elected Parallel Hash participant to\rallocate a hash table.
HashBatchElect Waiting to elect a Parallel Hash participant to allo-\rcate a hash table.
HashBatchLoad Waiting for other Parallel Hash participants to fin-\rish loading a hash table.
HashBuildAllocate Waiting for an elected Parallel Hash participant to\rallocate the initial hash table.
HashBuildElect Waiting to elect a Parallel Hash participant to allo-\rcate the initial hash table.
HashBuildHashInner Waiting for other Parallel Hash participants to fin-\rish hashing the inner relation.
HashBuildHashOuter Waiting for other Parallel Hash participants to fin-\rish partitioning the outer relation.
HashGrowBatchesAllocate Waiting for an elected Parallel Hash participant to\rallocate more batches.
HashGrowBatchesDecide Waiting to elect a Parallel Hash participant to de-\rcide on future batch growth.
HashGrowBatchesElect Waiting to elect a Parallel Hash participant to allo-\rcate more batches.
HashGrowBatchesFinish Waiting for an elected Parallel Hash participant to\rdecide on future batch growth.
HashGrowBatchesRepartition Waiting for other Parallel Hash participants to fin-\rish repartitioning.
