LWLock Wait Event Description
SInvalWrite Waiting to add a message to the shared catalog in-\rvalidation queue.
SubtransBuffer Waiting for I/O on a sub-transaction SLRU buffer.
SubtransSLRU Waiting to access the sub-transaction SLRU cache.
SyncRep Waiting to read or update information about the\rstate of synchronous replication.
SyncScan Waiting to select the starting location of a synchro-\rnized table scan.
TablespaceCreate Waiting to create or drop a tablespace.
TwoPhaseState Waiting to read or update the state of prepared\rtransactions.
WALBufMapping Waiting to replace a page in WAL buffers.
WALInsert Waiting to insert WAL data into a memory buffer.
WALWrite Waiting for WAL buffers to be written to disk.
WrapLimitsVacuum Waiting to update limits on transaction id and mul-\rtixact consumption.
XactBuffer Waiting for I/O on a transaction status SLRU\rbuffer.
XactSLRU Waiting to access the transaction status SLRU\rcache.
XactTruncation Waiting to execute pg_xact_status or update\rthe oldest transaction ID available to it.
XidGen Waiting to allocate a new transaction ID.
