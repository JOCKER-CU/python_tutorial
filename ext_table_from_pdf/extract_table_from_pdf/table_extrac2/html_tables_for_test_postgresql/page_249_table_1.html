Phase Description
initializing The command is preparing to begin scanning the heap. This phase is\rexpected to be very brief.
acquiring sample rows The command is currently scanning the table given by relid to\robtain sample rows.
acquiring inherited\rsample rows The command is currently scanning child tables to obtain sam-\rple rows. Columns child_tables_total, child_ta-\rbles_done, and current_child_table_relid contain the\rprogress information for this phase.
computing statistics The command is computing statistics from the sample rows obtained\rduring the table scan.
computing extended\rstatistics The command is computing extended statistics from the sample\rrows obtained during the table scan.
finalizing analyze The command is updating pg_class. When this phase is complet-\red, ANALYZE will end.
