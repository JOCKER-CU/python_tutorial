Some function calls can be simplified during planning based on properties specific to the function. For
example, int4mul(n, 1) could be simplified to just n. This type of transformation can be performed
by a planner support function, by having it implement the SupportRequestSimplify request type.
The support function will be called for each instance of its target function found in a query parse tree. If
it finds that the particular call can be simplified into some other form, it can build and return a parse tree
