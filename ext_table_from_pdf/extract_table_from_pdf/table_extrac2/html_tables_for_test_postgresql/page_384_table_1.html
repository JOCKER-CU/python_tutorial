The CREATE STATISTICS command has two basic forms. The first form allows univariate statistics for
a single expression to be collected, providing benefits similar to an expression index without the overhead
of index maintenance. This form does not allow the statistics kind to be specified, since the various statistics
kinds refer only to multivariate statistics. The second form of the command allows multivariate statistics
on multiple columns and/or expressions to be collected, optionally specifying which statistics kinds to
include. This form will also automatically cause univariate statistics to be collected on any expressions
