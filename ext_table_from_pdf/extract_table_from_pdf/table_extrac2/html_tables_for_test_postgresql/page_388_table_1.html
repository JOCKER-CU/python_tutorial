If a schema name is given then the text search template is created in the specified schema. Otherwise it
is created in the current schema.
You must be a superuser to use CREATE TEXT SEARCH TEMPLATE. This restriction is made because an
erroneous text search template definition could confuse or even crash the server. The reason for separating
templates from dictionaries is that a template encapsulates the “unsafe” aspects of defining a dictionary.
