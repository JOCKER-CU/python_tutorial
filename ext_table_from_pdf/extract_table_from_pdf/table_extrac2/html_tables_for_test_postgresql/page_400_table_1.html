pg_basebackup makes an exact copy of the database cluster's files, while making sure the server is put
into and out of backup mode automatically. Backups are always taken of the entire database cluster; it is
not possible to back up individual databases or database objects. For selective backups, another tool such
as pg_dump must be used.
The backup is made over a regular PostgreSQL connection that uses the replication protocol. The connec-
tion must be made with a user ID that has REPLICATION permissions (see Section 22.2) or is a supe-
