oprright oid (references pg_type.oid)\rType of the right operand
oprresult oid (references pg_type.oid)\rType of the result (zero for a not-yet-defined “shell” operator)

oprnegate oid (references pg_operator.oid)\rNegator of this operator (zero if none)
