useful for recovering from that situation.
Because of this convention of possibly renumbering OIDs assigned by patches, the OIDs assigned by a
patch should not be considered stable until the patch has been included in an official release. We do not
change manually-assigned object OIDs once released, however, as that would create assorted compatibility
problems.
