Escape Description
\d matches any digit, like [[:digit:]]
\s matches any whitespace character, like\r[[:space:]]
\w matches any word character, like [[:word:]]
\D matches any non-digit, like [^[:digit:]]
\S matches any non-whitespace character, like\r[^[:space:]]
\W matches any non-word character, like\r[^[:word:]]
