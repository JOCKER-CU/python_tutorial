0 counter-\nThis is the name of the counter that needs to be created or incremented or \nname\nprinted. It can be any custom name as the developer wishes.
1 integer This integer is an optional value that when provided next to the counter name \nwill represent the initial value of the counter (in counter-set, counter-reset \nproperties) or the value by which the counter should be incremented (in \ncounter-increment).
2 This is the initial value for all 3 counter-* properties. When this value is used \nnone\nfor counter-increment, the value of none of the counters are affected. When this \nis used for the other two, no counter is created.
3 counter-\nstyle This specifies the style in which the counter value needs to be displayed. It \nsupports all values supported by the list-style-type property. If none is used \nthen the counter value is not printed at all.
4 connector-\nThis represents the string that must be placed between the values of two \nstring\ndifferent counter levels (like the "." in "2.1.1").
